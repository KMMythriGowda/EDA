# Exploratory Data Analysis (EDA) Project

## Student Performance Dataset Analysis

---

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Student Performance dataset to uncover meaningful patterns, trends, and relationships between different student-related factors and academic performance.

The analysis uses statistical summaries and data visualizations to better understand the dataset and identify key influencing factors affecting student scores.

---

## Objectives

The main objectives of this project are:

- Analyze the student performance dataset
- Generate statistical summaries
- Visualize data distributions and trends
- Identify correlations between numerical variables
- Extract meaningful insights using EDA techniques
- Develop analytical thinking and data exploration skills

---

## Dataset Description

The dataset contains student-related information such as:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

The dataset is used to analyze academic performance trends and relationships between different educational and demographic factors.

## Key Insights

- Reading and writing scores show strong positive correlation
- Most students scored between 60 and 80
- Female students performed better in reading and writing
- Male students performed slightly better in mathematics
- Students completing test preparation courses performed better overall
- The dataset contains no missing values
